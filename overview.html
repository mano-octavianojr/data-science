Data Science

  • Instructor: Manolito Octaviano Jr.
  • Synchronous: TBA | Asynchronous: TBA

This course explores the transformative power of data science in enabling data-driven decision-making across today's dynamic enterprises. Building on foundational principles of business analytics, the curriculum expands into modern data science techniques to equip learners with essential tools and methodologies.

Participants begin with core concepts and progressively develop skills in data analysis and visualization using

Excel, SQL, and Python. Through hands-on projects, learners gain practical experience in data mining and machine learning, focusing on regression and classification models. The course also emphasizes critical thinking in analytics, including the distinction between correlation and causation, empowering students to derive meaningful insights and make strategic, evidence-based decisions.

Course objectives

  • Apply data analysis tools effectively
  • Visualize insights using Excel and Python
  • Differentiate correlation from causation clearly

Coursework

  • Programming (30%)
  • Project (70%)
