The data science course recognizes its pivotal role in driving data-informed decision-making across modern enterprises. While rooted in core business analytics principles, the curriculum expands into the realm of data science to equip learners with cutting-edge tools and methodologies. It begins with foundational concepts and advances through data analysis and visualization using Excel, SQL, and Python. Learners gain hands-on experience in data mining and machine learning for regression and classification. The course also addresses critical analytical concepts such as correlation versus causation, ensuring participants can extract meaningful insights and make strategic decisions grounded in data.

Course objectives

  • Apply data analysis tools effectively
  • Visualize insights using Excel and Python
  • Differentiate correlation from causation clearly

Coursework

  • Programming (30%)
  • Project (70%)
